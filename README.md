# Restaurant Delivery Management System

A simple Java Swing project developed using NetBeans. The system provides a graphical user interface for managing restaurant delivery orders.

## Features
- Add delivery orders
- Store customer information
- Store phone number and address
- Select food items
- Calculate the total price
- Simple graphical user interface
- About window

## Technologies Used
- Java
- Java Swing
- NetBeans

## Project Files
- `src/`: contains the Java source code
- `nbproject/`: contains NetBeans project settings
- `build.xml`: NetBeans build file
- `manifest.mf`: project manifest file
- `Order.txt`: stores order data

## How to Run
1. Open the project in NetBeans
2. Open the main class `RestaurantDelivery.java`
3. Run the project
